Coding Test: Find the Largest Stable Number | з¬”иҜ•пјҡ еҜ»жүҫжңҖеӨ§зҡ„зЁіе®ҡж•°

Problem йўҳйқў

Give a number $N$, find the largest number $k$ satisfy the following requirements smaller than $N$:

  1. Sum of all digits of $k$ is prime
  2. Each digits of $k$ is larger than or equals to the previous digit (e.g. 123456, 111111)

з»ҷе®ҡдёҖдёӘдёҠйҷҗ$N$, еҜ»жүҫжңҖеӨ§зҡ„е°ҸдәҺ$N$зҡ„ж•°еӯ—$k$пјҢиҰҒжұӮж»Ўи¶ід»ҘдёӢжқЎд»¶пјҡ

  1. $k$зҡ„еҗ„дёӘж•°дҪҚд№Ӣе’ҢжҳҜзҙ ж•°
  2. $k$зҡ„жҜҸдёҖдҪҚйғҪжҜ”еүҚдёҖдҪҚеӨ§ (жҜ”еҰӮ 123455, 11111)

Input иҫ“е…Ҙ

One line with one number $N$

дёҖиЎҢпјҢдёҖдёӘж•°еӯ—$N$

Output иҫ“еҮә

One line with one number $k$ satisfy the requirements or $-1$ if there is no solution

дёҖиЎҢпјҢдёҖдёӘж»Ўи¶іжқЎд»¶зҡ„ж•°еӯ—$k$пјҢеҰӮжһңжІЎжңүз¬ҰеҗҲжқЎд»¶зҡ„зӯ”жЎҲиҫ“еҮә$-1$

Size and Limits ж•°жҚ®и§„жЁЎе’ҢйҷҗеҲ¶

$N \leq 10^{18}$

Time Limits ж—¶й—ҙйҷҗеҲ¶: 200ms

Solution йўҳи§Ј

Obviously, the simplest approach is to try each number from $N$ down to $1$; if it satisfies the condition, return it, otherwise return $-1$. However, given the data scale up to $10^{18}$, this approach will definitely time out. By observing that the number of satisfying numbers is much smaller than the total range, we use a reverse thinking: generate numbers that meet the conditions via a search algorithm.

жҳҫз„¶пјҢжңҖз®ҖеҚ•зҡ„еҒҡжі•е°ұжҳҜд»Һ$N$еҲ°$1$дёҖдёӘдёӘиҜ•иҝҮеҺ»пјҢжүҫеҲ°дәҶе°ұиҝ”еӣһзӯ”жЎҲпјҢжІЎжүҫеҲ°е°ұиҝ”еӣһ$-1$гҖӮдҪҶиҖғиҷ‘еҲ°$10^{18}$зҡ„ж•°жҚ®и§„жЁЎпјҢиҝҷдёӘеҒҡжі•дёҖе®ҡдјҡи¶…ж—¶гҖӮйҖҡиҝҮи§ӮеҜҹеҸҜд»Ҙз®ҖеҚ•еҫ—еҲ°пјҢж»Ўи¶іжқЎд»¶зҡ„ж•°зҡ„и§„жЁЎиҝңе°ҸдәҺдёҠйҷҗеҶ…жүҖжңүж•°зҡ„и§„жЁЎпјҢжүҖд»ҘйҮҮеҸ–йҖҶеҗ‘жҖқз»ҙпјҢд»ҺжқЎд»¶е…ҘжүӢз”Ёжҗңзҙўз®—жі•з”ҹжҲҗж»Ўи¶іжқЎд»¶зҡ„ж•°еӯ—гҖӮ

First list all possible primes via the sieve method

йҰ–е…ҲпјҢйҖҡиҝҮзӯӣжі•еҲ—еҮәжүҖжңүеҸҜиғҪзҡ„зҙ ж•°пјҡ

For each prime, find the largest number with non-decreasing digits and digit-sum equal to that prime

з„¶еҗҺпјҢеҜ№дәҺжҜҸдёҖдёӘзҙ ж•°пјҢжҲ‘д»¬иҰҒжүҫеҲ°жңҖеӨ§зҡ„ж»Ўи¶іжқЎд»¶2дё”е’ҢдёәеҪ“еүҚиҝҷдёӘзҙ ж•°зҡ„ж•°еӯ—гҖӮз„¶еҗҺеҶҚиҝҷдәӣж•°еӯ—йҮҢйқўжүҫеҲ°жңҖеӨ§зҡ„еҖјиҝӣиЎҢиҝ”еӣһ

Implement the DFS function to build the largest valid number

  1. Convert the number to a digit array for easy generation.
  2. If the sum of digits so far exceeds the target prime, prune this branch.
  3. If all digits are assigned and the remaining sum is zero, return the constructed number.
  4. Generate digits from most significant to least significant, iterating from the maximum allowable digit down to the minimum to ensure the first found solution is the largest.
  5. Determine the upper bound for the current digit: if all previous digits match the corresponding digits of $N$, the upper bound is that digit of N; otherwise it is $9$.
  6. Determine the lower bound for the current digit: it must be at least the previous digit to maintain non-decreasing order (or $0$ if itвҖҷs the first digit).
  7. For the current position, try all digits in [lower_bound, upper_bound] in descending order; if a recursive call returns a valid number, return it immediately.
  8. If no digit works, return -1.

жңҖеҗҺпјҢжҲ‘д»¬жқҘе…·дҪ“е®һзҺ°еҜ№дәҺжҜҸдёҖдёӘзҙ ж•°пјҢжүҫеҲ°жңҖеӨ§зҡ„ж»Ўи¶іжқЎд»¶зҡ„ж•°еӯ—зҡ„dfsеҮҪж•°пјҡ

  1. йҰ–е…ҲжҠҠж•°еӯ—иҪ¬еҢ–дёәеҜ№еә”зҡ„ж•°дҪҚж•°з»„пјҢж–№дҫҝз”ҹжҲҗж•°еӯ—гҖӮ
  2. еҰӮжһңеҪ“еүҚж•°дҪҚж•°з»„д№Ӣе’Ңе·Із»Ҹи¶…иҝҮжҢҮе®ҡзҡ„зҙ ж•°пјҢиҜҙжҳҺеҪ“еүҚзҡ„dfsж–№еҗ‘е·Із»ҸдёҚеӯҳеңЁи§ЈпјҢйҖҖеӣһеҲ°дёҠдёҖеұӮгҖӮ
  3. еҰӮжһңе·Із»Ҹе®ҢжҲҗдәҶжүҖжңүдҪҚзҡ„з”ҹжҲҗпјҢдё”е’ҢеҲҡеҘҪзӯүдәҺжҢҮе®ҡзҡ„зҙ ж•°пјҢйӮЈд№ҲеҲҷжүҫеҲ°зӯ”жЎҲе№¶иҝ”еӣһи§ЈгҖӮ
  4. д»Һй«ҳдҪҚеҲ°дҪҺдҪҚдёҖдҪҚдҪҚз”ҹжҲҗпјҢеҜ№дәҺжҜҸдёҖдҪҚпјҢд»ҺиҢғеӣҙеҶ…жңҖеӨ§зҡ„ж•°еӯ—еҫҖиҢғеӣҙеҶ…жңҖе°Ҹзҡ„ж•°еӯ—йҒҚеҺҶпјҢдҝқиҜҒжүҫеҲ°зҡ„з¬¬дёҖдёӘз»“жһңе°ұжҳҜжңҖеӨ§зҡ„з»“жһңгҖӮ
  5. еҲӨж–ӯеҪ“еүҚдҪҚж•°еӯ—зҡ„дёҠз•ҢпјҡеҰӮжһңд№ӢеүҚжүҖжңүдҪҚзҡ„ж•°еӯ—йғҪзӯүдәҺ$N$зҡ„еҜ№еә”дҪҚпјҢйӮЈд№ҲеҰӮжһңеҪ“еүҚдҪҚеӨ§дәҺдәҶ$N$зҡ„еҜ№еә”дҪҚпјҢе°ұдјҡи¶…иҝҮиҢғеӣҙпјҢжүҖд»Ҙиҝҷз§Қжғ…еҶөдёӢдёҠйҷҗдёә$N$зҡ„еҜ№еә”дҪҚзҡ„ж•°еӯ—пјӣеҰӮжһңд№ӢеүҚе·Із»ҸжңүжҹҗдёҖдҪҚзҡ„ж•°еӯ—е°ҸдәҺ$N$зҡ„еҜ№еә”дҪҚпјҢйӮЈд№Ҳж— и®әеҪ“еүҚдҪҚзҡ„ж•°еӯ—еӨ§е°ҸжҳҜеӨҡе°‘пјҢиҜҘж•°еӯ—йғҪдёҚеҸҜиғҪи¶…иҝҮиҢғеӣҙпјҢжүҖд»ҘдёҠйҷҗдёәеҚ•дёӘдҪҚзҡ„жңҖеӨ§еҖј$9$гҖӮ
  6. еҲӨж–ӯеҪ“еүҚдҪҚж•°еӯ—зҡ„дёӢз•Ңпјҡйўҳзӣ®иҰҒжұӮеҪ“еүҚдҪҚеӨ§дәҺзӯүдәҺеүҚдёҖдҪҚпјҢжүҖд»ҘдёӢз•Ңе°ұжҳҜеүҚдёҖдҪҚзҡ„ж•°еӯ—гҖӮ
  7. еҜ№дәҺеҪ“еүҚдҪҚд»ҺеӨ§еҲ°е°ҸйҒҚеҺҶиҢғеӣҙеҶ…жүҖжңүеҖјпјҢеҰӮжһңжүҫеҲ°зӯ”жЎҲзӣҙжҺҘиҝ”еӣһеҖјйҖҖеҮәгҖӮ
  8. еҰӮжһңжІЎжңүжүҫеҲ°и§ЈпјҢиҝ”еӣһ-1
